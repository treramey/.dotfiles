| Keybind      | Description                   |
| ------------ | ----------------------------- |
| `<leader>dd` | Start debugging               |
| `<leader>dS` | Stop debugging                |
| `<leader>dc` | Continue debugging            |
| `<leader>dp` | Pause debugging               |
| `<leader>dr` | Run without debugging         |
| `<leader>dR` | Restart debugging             |
| `<leader>di` | Step into                     |
| `<leader>ds` | Step over                     |
| `<leader>do` | Step out                      |
| `<leader>db` | Toggle breakpoint             |
| `<leader>dB` | Toggle inline breakpoint      |
| `<leader>dj` | Jump to cursor                |
| `<leader>dv` | Toggle REPL                   |
| `<leader>dw` | Focus on watch window         |
| `<leader>dW` | Add to watch                  |
| `<C-[>`      | Go to previous buffer         |
| `<C-]>`      | Go to next buffer             |
| `leader r`   | Rename                        |
| `leader v`   | Vertical split                |
| `leader s`   | Horizontal split              |
| `U`          | Redo                          |
| `H`          | Go to start of line           |
| `L`          | Go to end of line             |
| `leader d`   | Show hover                    |
| `leader w`   | Save                          |
| `leader q`   | Quit                          |
| `leader x`   | Save and quit                 |
| `[d`         | Go to previous marker         |
| `]d`         | Go to next marker             |
| `<leader>ca` | Quick fix                     |
| `leader sf`  | Quick open                    |
| `<leader>sg` | Find within files             |
| `leader f`   | Format document               |
| `gh`         | Show definition preview hover |
